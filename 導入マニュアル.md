# 健保問い合わせサポートシステム　導入マニュアル（ご利用者向け）

このマニュアルは、本システムをお使いになる**健保組合のご担当者さま**向けの手引きです。
専門知識がなくても進められるよう、順番に説明します。

> 所要時間の目安：**初回のみ 20〜40分**（AIの準備を含む）。2回目以降はダブルクリックで起動するだけです。

---

## ＜全体の流れ＞

```
① 本システム（kenpo-support）を起動する
        ↓
② 使うAIを1つ決める（Claude / ChatGPT / Gemini）
        ↓
③ 画面の「⚙ 初期設定」で、導入→ログイン→接続確認 ←初回だけ
        ↓
④ 画面右上で使うAIを選んで、利用開始
```

> 💡 AIの準備（③）は、**画面の「初期設定」ボタンからボタン操作で行えます**。
> コマンド入力は不要です（ログインの「許可」操作のみご自身で行います）。

---

## ① 本システムを起動する

1. お渡しした **`kenpo-support.exe`**（Windows）を**ダブルクリック**します
   - macの場合は `kenpo-support` をダブルクリック
2. 小さな黒い画面（起動状況の表示）が出て、**自動的にブラウザで画面が開きます**
   - 開かない場合は、ブラウザのアドレス欄に `http://127.0.0.1:8765/` と入力
3. 使い終わったら、黒い画面で **Ctrl + C** を押すか、画面を閉じれば終了します

> ⚠️ Windowsで「WindowsによってPCが保護されました」と出た場合は、
> 「詳細情報」→「実行」で起動できます（社内ルールに従ってください）。

---

## ② 使うAIを1つ決める

本システムは以下の3つのAIに対応しています。**すでに有料プランをご契約のもの**を選ぶのがおすすめです。

| AI | 対応する契約の例 |
|----|------------------|
| **Claude**（クロード） | Claude Pro / Max |
| **ChatGPT**（チャットジーピーティー） | ChatGPT Plus / Pro |
| **Gemini**（ジェミニ） | Google アカウント（Gemini） |

> 後から画面で切り替えられます。まずは1つで始めれば十分です。

### 共通の準備：Node.js を入れる（初回のみ）

AIの導入には「Node.js」というソフトが必要です。未導入の場合のみ、先に入れてください。

- 公式サイト <https://nodejs.org/> から「LTS」版をダウンロードして、画面に従いインストール
- インストール後、本システムを一度再起動してください
- （初期設定画面に「✅ Node.js 導入済み」と表示されていれば不要です）

---

## ③ 「⚙ 初期設定」でAIを準備する（初回だけ・コマンド入力不要）

画面右上の **「⚙ 初期設定」** ボタンを押すと、準備用の画面が開きます。
選んだAIの欄で、上から順にボタンを押すだけです。

1. **① 導入する** … クリックするとAIのプログラムが自動で入ります（数分かかることがあります）
2. **② ログイン** … ボタンを押すと別画面が開きます。表示された案内に従い、
   ブラウザで**ご自身のアカウントにログイン／「許可」**してください
3. **③ 接続確認** … 「✅ 接続できました」と出れば準備完了です

> - 利用するAIだけ準備すればOKです（1つで始められます）。
> - 一度ログインすれば、次回以降は不要です。
> - うまくいかない場合は、社内の情報システム担当の方にご相談ください。

---

## （参考）コマンドで手動準備する場合

「初期設定」ボタンを使わず、手動で準備することもできます（情シス担当者向け）。
コマンドプロンプト（macは「ターミナル」）で、下表のコマンドを実行します。

| AI | 導入 | ログイン |
|----|------|----------|
| **Claude** | `npm install -g @anthropic-ai/claude-code` | `claude auth login` |
| **ChatGPT** | `npm install -g @openai/codex` | `codex login` |
| **Gemini** | `npm install -g @google/gemini-cli` | `gemini`（初回起動で認証） |

---

## ④ 使い方

1. 画面**右上の「AI」**で、準備したAIを選びます
   - 準備していないAIは「（未導入）」と表示され、選べません
2. **新規問い合わせ**：カテゴリ・担当者名・問い合わせ内容を入力し、
   「**AI回答草案を生成**」を押します
   → 回答文と、根拠となる参考文献（法令・条文）が表示されます
3. **AIと相談**：草案について不明点を質問できます
4. **最終回答へ反映**：相談結果をふまえ、回答を仕上げます
5. **保存／履歴**：「対応中」「完了」で保存。「対応履歴」から検索・編集できます

---

## データの保存場所

- 問い合わせ・回答・履歴・設定は、**お使いのPCの中だけ**に保存されます
  （保存先：`ユーザーのフォルダ\.kenpo-support\kenpo_support.db`）
- 外部のサーバーには保存されません

> 📌 **AIへの送信について**：回答を作る際、問い合わせの本文は選んだAI会社
> （Anthropic／OpenAI／Google）のクラウドに送信されます。被保険者の個人情報の
> 取り扱いは、貴組合の規程に従ってご利用ください。

## 困ったときは（トラブルシューティング）

| 症状 | 対処 |
|------|------|
| 右上でAIが「（未導入）」になっている | 「⚙ 初期設定」で③の導入・ログインが完了しているか確認。完了後、本システムを再起動 |
| 「CLIが見つかりません」と出る | そのAIが未導入。「⚙ 初期設定」の「① 導入する」を実行 |
| 「ログイン状態・利用上限を確認」と出る | 「⚙ 初期設定」の「② ログイン」をやり直す。または利用上限の可能性 |
| 応答が返ってこない／遅い | 少し待つ。AI側が混雑している場合があります |
| ブラウザ画面が開かない | アドレス欄に `http://127.0.0.1:8765/` を直接入力 |
